개인정보처리방침

시행일: 2026년 6월 12일

요약부터. 아파트핏은 계정도 가입도 없습니다. 운동 기록, 몸무게, 무게 설정 같은 데이터는 전부 사용자의 기기 안에만 저장되고, 우리 서버로 전송되지 않습니다. 서버에 닿는 건 딱 두 가지 — 직접 참여를 선택한 벤치마크 기록과, 익명 사용 통계뿐입니다.

1. 수집하지 않는 것

이름, 이메일, 전화번호, 위치, 연락처, 사진 등 개인을 식별할 수 있는 정보를 수집하지 않습니다. 운동 기록·신체 정보는 기기의 로컬 저장소(localStorage)에만 보관되며, 앱을 삭제하면 함께 사라집니다. 백업 파일은 사용자가 직접 내보내고 보관합니다.

2. 수집하는 것

항목내용시점
익명 사용 통계 무작위 익명 ID, 화면 이동·기능 사용 이벤트, 기기·브라우저 종류. 운동 기록 수치는 포함되지 않습니다. 앱 사용 중 (설정에서 끌 수 있음)
벤치마크 기록 직접 입력한 별명, 제출한 기록 수치, 익명 ID "단지 벤치마크"에 직접 제출할 때만
친구 초대 확인 초대한 쪽·받은 쪽의 익명 ID, 중복 방지용 IP 해시(원본 IP는 저장하지 않음) 초대 링크로 시작할 때만

3. 처리 위탁 및 국외 이전

위 데이터의 저장·처리를 위해 아래 서비스를 이용하며, 서버는 미국에 있습니다.

  • PostHog Inc. — 익명 사용 통계 분석
  • Cloudflare, Inc. — 벤치마크·초대 API 운영
  • Neon, Inc. — 벤치마크 기록 데이터베이스

4. 보유 기간과 파기

벤치마크 기록은 앱 안의 삭제 버튼으로 언제든 지울 수 있고, 삭제 즉시 서버에서 제거됩니다. 익명 통계는 서비스 개선 목적 달성 후 파기합니다. 서비스를 종료하면 모든 서버 데이터를 파기합니다.

5. 이용자의 권리

수집되는 게 익명 데이터뿐이라 계정 탈퇴 절차 자체가 없습니다. 그래도 본인 데이터 삭제를 원하시면 앱 내 삭제 기능을 쓰시거나 아래 연락처로 요청해 주세요. 지체 없이 처리합니다.

6. 만 14세 미만

아파트핏은 만 14세 미만 아동의 개인정보를 수집하지 않습니다.
